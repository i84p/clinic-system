Clinic Management System
📌 نبذة عن المشروع

نظام داخلي لإدارة العيادات يهدف إلى تنظيم المواعيد وإدارة بيانات المرضى بين موظف الاستقبال والطبيب، مع تحسين سرعة وكفاءة العمل داخل العيادة وتقليل الازدحام.

🎯 فكرة النظام

النظام يساعد على:

حجز وتنظيم المواعيد بدون تعارض
إدارة بيانات المرضى وسجلهم الطبي
متابعة حالة المريض داخل العيادة
إصدار الفواتير بعد الجلسات
إرسال إشعارات وتنبيهات للمواعيد
👥 المستخدمين
موظف الاستقبال (Receptionist)
الطبيب (Doctor)
⚙️ مميزات النظام
تسجيل وإدارة المرضى
حجز وتعديل وإلغاء المواعيد
إدارة حالة المريض (Waiting, In Progress, Completed, No-show)
إدارة الجلسات الطبية والتشخيص
عرض السجل الطبي للمريض
إنشاء الفواتير وتتبع الدفع
تقارير يومية وإشعارات تلقائية
🔄 طريقة العمل
تسجيل المريض
حجز موعد
دخول العيادة (Waiting Room)
بدء الجلسة مع الطبيب
إنهاء الجلسة
إصدار فاتورة
الدفع وإغلاق الحالة
📝 ملاحظات
النظام للاستخدام الداخلي فقط
المريض لا يستخدم النظام مباشرة
التركيز على البساطة والسرعة في الأداء
